- **~70 assets** dont **~45 générés par IA** (B3-B8, B12, C2, C4-C5, D1-D3 : familles à prompts) — le reste vient de l'API, du pipeline ou du design system ;

- Nommage : `media/{famille}/{slug}@{taille}.{ext}` versionné dans le repo.
