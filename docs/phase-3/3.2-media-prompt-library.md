### 1.3 Adaptations par modèle (D21 §6)
| Modèle | Adaptation |
|---|---|
| **Flux / GPT Image / ChatGPT Images** | Prompt tel quel (langage naturel) |
| **Midjourney** | Condenser aux 15-25 mots à plus fort signal + `--ar {ratio}` + `--style raw` ; négatifs via `--no text, watermark, celebrity` |
| **Ideogram** | Réservé aux assets **avec texte** (§6-7) : ajouter `TEXT: "{texte exact}"` et style de lettrage ; CFG élevé |

### 1.4 Règles de production (rappel D21/D22)
Ratio déclaré avant génération · rendu 2× la taille cible · export AVIF/WebP · zone de texte laissée vide sauf Ideogram · itérer par lots de 4, garder le meilleur, réutiliser la seed pour la famille.
